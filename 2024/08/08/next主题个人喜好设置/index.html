5、Local Search本地搜索

5.1 安装插件hexo-generator-searchdb,执行以下命令:

npm install hexo-generator-searchdb --save

5.2 修改博客的配置文件，新增以下内容到任意位置：

search:
path: search.xml
field: post
format: html
limit: 10000 #最多显示搜索结果的数量为10000个

5.3 编辑 主题配置文件，启用本地搜索功能：

打开主题的配置文件，把# local search的enable改成true。
