Python编程环境搭建

Python的编程环境搭建，只需要两个条件：

  • 代码编辑器
  • Python解释器

代码编辑器：理论上任何能编辑文本的软件都可以充当代码编辑器。但这里，我推荐使用VSCode作为代码编辑器。

Python解释器：因为Python是高级编程语言，需要相应的解释器才能将代码转换为机器码。人类能看懂代码，但是机器只能认识0和1。这里直接去Python官网下载解释器即可。

以下是具体步骤

1、下载vsCode

打开vsCode官网，点击Download下载vsCode。

若无法打开官网，也可点击这里进行下载。（建议去官网下载）备用下载vsCode

2、下载Python

打开Python官网，下载Python解释器。

备用下载

3、安装Python解释器

1️⃣双击打开刚刚下载的vsCode安装包，会看到如下界面，按照我图片里面的提示进行操作。

  • 点击Customize install，开始设置Python解释器的安装位置。
  • 勾选上Add python.exe to PATH

2️⃣勾选上for all users 意思是为 所有用户安装

3️⃣点击Browse，设置安装位置。

不建议安装在C盘，因为啥都安装在C盘，用不了多久C盘就爆满啦！

4️⃣设置好了安装位置后，点击Install，开始安装。

出现如下界面就是安装完成啦！

点击右上角关闭界面，或者点击右下角Close关闭界面即可。

4、安装并配置vsCode

1️⃣双击打开下载好的vsCode安装包，先勾选我同意此协议，再点击下一步。

2️⃣勾选上创建桌面快捷方式，点击下一步。

3️⃣点击安装

4️⃣点击完成，此时vsCode已经安装好了。

5️⃣打开vsCode软件，界面如下。

英文界面，用不习惯，接下来进行汉化。

6️⃣点击图片所示按钮（或者按下ctrl+shift+x键），打开扩展商店。

7️⃣在搜索框输入简体中文，点击排名第一的那个右下角的Install安装中文包。

8️⃣点击右下角Change Language and Restart按钮，进行重启。

9️⃣在vsCode中安装python插件

继续打开vsCode的扩展商店，搜索python，安装python插件。

这个插件比较大，安装大约需要1~2分钟，具体时间取决于你的网速。

安装完成之后，至此，咱们的Python环境就算搭建完成啦！

文件名必须要以.py结尾。
